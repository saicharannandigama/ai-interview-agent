# 🤖 AI Interview Agent

An AI-powered interview practice platform where users can:
- Practice interview questions
- Get AI-generated responses
- Improve communication skills

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- OpenAI API

## 📦 Features
- Real-time chat interface
- AI-based interview responses
- Clean modern UI

## ▶️ Run Locally

### Frontend
npm install
npm start

### Backend
cd backend
npm install
node server.js
